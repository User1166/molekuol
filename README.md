
# Molekuol

Moleküler yapıları görselleştirmek ve analiz etmek için geliştirilmiş bir web uygulaması.

## Özellikler

- **3D Moleküler Görselleştirme:** Moleküllerin üç boyutlu modellerini görüntüleyin.
- **Renklendirme Özellikleri:** Molekülleri farklı özelliklere göre renklendirin.
- **Veritabanı İşlevleri:** Moleküler verileri yönetmek için veritabanı entegrasyonu.

## Teknoloji Yığını

- **Python:** Sunucu tarafı geliştirme için.
- **HTML:** Uygulamanın yapısı için.
- **CSS:** Stil ve düzen için.
- **JavaScript:** Dinamik içerik ve etkileşim için.
